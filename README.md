# Wizard-slider

Слайдер на Vue.js

Деплой: https://myway778.github.io/wizard-slider/

## Установка проекта

```sh
npm install
```

### Запуск dev сервера

```sh
npm run dev
```

### Production сборка

```sh
npm run build
```

### Запуск ESLint

```sh
npm run lint
```

### Форматирование

```sh
npm run format
```
